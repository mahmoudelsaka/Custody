Elsaka Custody



Overview,



This module will help HR manager to manage employees custody and tracking devices





Custody Work-flow

1- New: In case of new request to issue a equipment for specific employee

2- In Progress: It will be status while issuing the device

3- Delivered: It will be status after delivering device or equipment to Employee

4- Closed: after Employee return the device to company or custody manger

5- Cancel : It will be status when order is canceled

Access Right

Custody User: can create request and convert it to progress only

Custody manager: Has full access to create, edit, delete any record,

Validation:-

Only available devices or equipment will be shown when user create custody request.

1-Item: you can configure Items which you need to be part of equipment (example: HDD, RAM) Define Name & ID for every Item

1-Equipment: You can define equipment specifications which will be used in custody request

1- Name: Enter Equipment Name (Example: Laptop E450)

2- Code & Serial Number: Enter Equipment Code, and Serial number,

Category: Select one from categories list which should be defined before (example: Lenove Laptops)

Then you can add equipment's items and its value

Example: (HDD: 500GB) & (RAM : 4GB)

Also you can add also internal notes
